# Employee Salary & Performance Analysis - EDA

## Project Overview

This project performs Exploratory Data Analysis on employee data to identify patterns, trends, and relationships.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

- Statistical summary
- Salary distribution analysis
- Department comparison
- Experience vs salary analysis
- Correlation analysis
- Employee performance analysis

## Conclusion

The project demonstrates the complete EDA workflow from data exploration to insight generation.
